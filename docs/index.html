From Geography to Development

Does economic structure mediate the latitude-income association? A cross-country analysis of 173 nations with machine learning classification.

Project Overview

Countries at higher latitudes are systematically richer than tropical nations. But is this a direct geographic effect, or does it operate through economic structures that geography shapes? This project uses 2019 cross-country data for 173 nations to test whether agriculture share, urbanisation, and trade openness mediate the latitude–income gap.

Methods Overview

Data

REST Countries API + World Bank Indicators (2019). Inner join on ISO3 codes; complete-case filter → 173 countries.

Feature Engineering

12 features: geographic (latitude², landlocked, tropical), structural (agriculture, urbanisation, trade), and interactions.

LASSO + OLS

LASSO (5-fold CV) selects 11 features. Five nested OLS models with HC3 robust SE track latitude coefficient attenuation as structure variables enter.

ML Classification

Stratified 80/20 split (Ntrain=138, Ntest=35). Logistic Regression, Random Forest, XGBoost — hyperparameters tuned by 5-fold CV.

Clustering

K-Means (K=2–8) on standardised structure variables. K selected by silhouette score.

Key Findings

🌎

Geography shapes structure

Distance from the equator correlates with lower agriculture dependence (r = -0.39) and higher urbanisation (r = 0.38), confirming geographic-structural linkages.

📉

Structure mediates ~51% of income gap

The latitude coefficient drops from 0.045 (geography only) to 0.022 after adding structure variables - a 51% attenuation - but remains statistically significant.

🌾

Agriculture is the dominant predictor

Agriculture share (r = -0.79 with log GDP) is the top feature in all three ML models. Urbanisation ranks second (r = 0.72).

🤖

ML classifies with high accuracy

Random Forest achieves AUC = 0.958 and F1 = 0.8462 on the held-out test set. All three classifiers exceed AUC = 0.92.
